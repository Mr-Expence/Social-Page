# Social-Page
A html page that adds 6 quick links to socials. You can change the social links to any of these premade styles but you can create your own.

There is support for:

| Platforms  | Limitations     |
|------------|-----------------|
| Discord    | None            |
| Youtube    | None            |
| Twitch     | None            |
| Modrinth   | None            |
| Curseforge | No profile data |
| Github     | None            |
| X/Twitter  | No profile data |
| Instagram  | None            |
| Reddit     | None            |
| Telegram   | No profile data |
